Exercice n°2



Le langage CSS permet aussi de manipuler des classes. Il s’agit de définir un style qui s’appliquera à tous les éléments qui auront cette classe dans leur paramètre.

Cela permet aussi d’appliquer un style différent à deux paragraphes par exemples.


I. Préparation des fichiers


1. Télécharger et sauvegarder dans votre dossier : mes documents/SNT/le web les deux  fichiers suivants :

oeuf.html et style_oeuf.css

2) Ouvrir le fichier oeuf.html dans le navigateur.

3) Ouvrir les fichiers oeuf.html et style_oeuf.css avec le logiciel Notepad++.


Répondez aux questions suivantes sur le document réponse : document réponse langage CSS exercice n°2



II. Premiers changement de style

1) D’après le fichier style_oeuf.css, comment devrait-être le titre du poème ?

2) Ce n’est pas le cas, pourquoi ? Appliquer les changements nécessaires.



III. Appliquer une classe

1) Le fichier style_oeuf.css décrit le style de la classe poème. Comment ce style est-il déclaré ?

2) Appliquons ce style en remplaçant la 1ère balise <div> par <div class="poeme"> dans le fichier oeuf.html

3) Actualiser la page oeuf.html dans le navigateur.Que c’est-il passé ?



IV. Créer des classes

1) Déclarer trois classes bleu,jaune et blanc dans le fichier style_oeuf.css qui fixent la couleur du texte en respectivement bleu, jaune et blanc.

2) Réaliser les changements nécessaires dans le fichier oeuf.html pour appliquer le style adéquat aux trois strophes.

3) Actualiser la page dans le navigateur pour vérifier vos changements.



V. Ajout d’une image

1) Ajouter une image d’œuf en dessous du poème.

2) Modifier le CSS pour arrondir les angles de cette image (vous ferez une recherche sur internet pour trouver l'instruction).
